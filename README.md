Nexa is a local AI Romantic assistant built in Python that supports
**text + voice interaction** and **persistent memory** using JSON.

Runs fully from the command line.

---

## Features
- CMD-based AI assistant
- Optional voice output (ElevenLabs)
- Local persistent memory (JSON)
- Simple, hackable architecture

---

## Requirements
- Python 3.9+
- OpenAI API key
- ElevenLabs API key (optional)
